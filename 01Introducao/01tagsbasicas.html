Para abrir um link em outra página, há o atributo target = "_blank"

Tipos de links

Se você está linkando para outras páginas dentro do mesmo site, não precisa especificar o link inteiro, dá para usar a forma abreviada href="Nome_do_arquivo.html" é suficiente. Já que esses arquivos estão na mesma página, fica fácil de linkar.

  • "http://....com.br": link externo, vai para um arquivo fora do meu site
  • "outraPágina.html": Cria um link para uma página no próprio site
  • "#secao": Serve para navegar por tópicos/títulos na mesma página
  • "mailto:contato@gmail.com": link para email. exige que o usuário possua software gerenciador de emails, tipo outlook. Não é comum.
  • Isso serve para linkar para outro site e ainda especifica qual seção do site deve ser mostrada logo que a pessoa entra. Para que isso funcione, a seção deve ter um id, e esse id é o que será especificado no link "#nome_da_seção".

Em grandes websites, é importante organizar cada página(arquivo html) por seções específicas. Ajuda quem constrói e também à linká-los. página 89 do livro, em caso de dúvidas.
