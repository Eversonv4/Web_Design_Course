Box-Sizing

Normalmente, a altura e largura de um elemento é calculada assim:

width + padding + border = largura total de um elemento
height + padding + border = altura total de um elemento

Essa propriedade define como a largura e altura de um elemento são calculadas: devem incluir preenchimento e bordas ou não.

Normalmente, quando especificamos a largura e altura de um componente e logo depois adicionamos um padding, as dimensões aumentam em todas as direções.

Por exemplo: Um componente com altura de 100px e largura de 50px, ao acrescentar um padding de 25px, o tamanho externo (das bordas) terá uma altura de 150px, pq foi adicionado 25px tanto pra cima quanto pra baixo, e a largura de 100px, pois foi adicionado 25px tanto à esquerda quanto à direita.

Usando Box-Sizing:

Ao usar a propriedade box-sizing: border-box; junto às outras propriedades de altura, largura e padding, você garante que os valores passados serão os valores reais do componente. O que ele faz é diminuir o tamanho do conteúdo central, se necessário. Para que se adeque às medidas passadas.

{ box-sizing: border-box | content-box | initial | inherit }

  • content-box Valor padrão. É o que normalmente o css faz, sem a necessidade de usar box-sizing. Ele soma os valores de padding acima dos valores de largura e altura, fazendo com que o componente em si seja maior que os valores de dimensões passados.
  • border-box O que ele faz é diminuir o tamanho do conteúdo central, se necessário. Para que se adeque às medidas passadas.

Ambas as divs(content-box e border-box) receberam um atributo padding de 25px: uma aumentou o tamanho do seu componente, mantendo tamanho do conteúdo intacto. A outra manteve o tamanho do seu componente igual, mas diminuiu as dimensões do seu conteúdo para se adequar às medidas passadas.
