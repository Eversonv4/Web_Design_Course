Propriedade Display

Exemplo: body { display: inline | block | inline-block | contents | none | tem mais no site ;}

inline

Ele divide os elementos da página em blocos. Ele funciona com se fosse uma única linha, vai colocando as coisas uma do lado da outra até pegar toda a largura da página ou container. Não se aplica propriedades de model box em elementos de linha.

inline-flex

Mostra um elemento como se fosse um objeto inline, serve para alinhar em linha objetos que normalmente são blocos. Tipo alinhar divs, que normalmente ficariam cada uma em seu bloco, em linhas diferentes

Everson

block

Também divide os elementos da página em blocos. Ele normalmente ocupa a área toda disponível(a largura toda a página ou de um container, width 100%) e vai gerar uma quebra de linha. Fazendo com que os elementos fiquem um em baixo do outro.

inline-block

O elemento se comporta como elemento de linha, mas com características de um elemento de bloco. Não gera quebra de linha.

contents

Faz com que o contêiner desapareça, tornando os elementos filhos, filhos do elemento no próximo nível acima no DOM

none

Essa propriedade retira da página o elemento selecionado.

Visibility

Essa propriedade esconde um elemento, ele continua lá no texto, ocupando espaço da página, mas não é possível visualizá-lo. fica só um espaço em branco onde esse elemento deveria estar.

Exemplo: body { visibility: hidden | visible | collapse | initial | inherit ;}

  • Hidden: Bastante intuitivo, ESCONDER ALGO.
  • Visible: Também bastante intuitivo, MOSTRAR ALGO QUE FOI/ESTÁ ESCONDIDO.
  • Collapse: Apenas para linhas da tabela (<'tr>), grupos de linhas (<'tbody>), colunas (<'col>), grupos de colunas (<'colgroup>). Este valor remove uma linha ou coluna, mas não afeta o layout da tabela. O espaço ocupado pela linha ou coluna ficará disponível para outros conteúdos. Se o collapse for usado em outros elementos, ele será processado como "oculto"

Usar *{ propriedade: valor } é um jeito CSS de marcar todos os elementos da página.

Ex: *{ display: inline-block ;}
