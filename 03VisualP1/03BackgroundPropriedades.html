CSS Background Propriedades:

  • background-attachment

    Define se uma imagem de background está fixada ou rola junto com o resto da página.

    • { background-attachment: scroll | fixed | local | initial | inherit ;}

    • Scroll: a imagem de fundo vai rolar junto com a página. isso é o padrão
    • Fixed: a imagem de fundo não vai rolar com a página
    • Local: a imagem de fundo vai rolar com os conteúdos dos elementos.
    • Initial: Define essa propriedade(bg-attachment) para o seu valor padrão.
    • Inherit: herda essa propriedade(bg-attachment) do seu elemento pai.

    Exemplo de como criar um efeito de rolagem de paralaxe simples (cria uma ilusão de profundidade 3D):

    .fixed-bg {
    | /* The background image */
    | background-image: url("img_tree.gif");
    |
    | /* Set a specified height, or the minimum height for the background image */
    | min-height: 500px;
    |
    | /* Set background image to fixed (don't scroll along with the page) */
    | background-attachment: fixed;
    |
    | /* Center the background image */
    | background-position: center;
    |
    | /* Set the background image to no repeat */
    | background-repeat: no-repeat;
    |
    | /* Scale the background image to be as large as possible */
    | background-size: cover;
    }

  • background-clip

    Especifica a área de pintura do background. Essa propriedade define até que ponto do background (color ou imagem) deve extender dentro de um elemento.

    • { background-clip: border-box | padding-box | content-box | initial | inherit ;}

    • Border-box: É o valor padrão. O background se extende por trás da borda e vai até ela.
    • Padding-box: O background se extende só até encostar na borda, preenche o conteúdo e a parte do padding.
    • Content-box: O background se extende só até as limitações. Ex: No caso de um texto, por mais que o componente fosse grande, ele só teria esse fundo nas delimitações do texto.
    • Initial: Define essa propriedade(bg-clip) para o seu valor padrão.
    • Inherit: herda essa propriedade(bg-clip) do seu elemento pai.

  • background-color

    Define a cor do background de algum elemento, não necessariamente da página.

    • { background-color: nome_da_cor | transparent | initial | inherit ;}

    Escrevendo cores: red | #00FF00 | rgb(25,122,255) | rgba(0,0,0, 0-1) | transparent.

  • background-image

    Define uma imagem para o background de um elemento.

    • { background-color: url("") | linear-gradient() | radial-gradient() | none | initial | inherit ;}

    • url("URL"): A url da imagem. Para especificar mais de uma imagem, precisa separar com uma VÍRGULA.
    • linear-gradient("direção", "color 1", "color 2", ...): Define um gradiente linear de cor como imagem de fundo. Tem que escolher pelo menos duas cores.

      • Ex:
      background-image: linear-gradient(red, yellow, blue);
      background-image: linear-gradient(to right, red , blue);
      background-image: linear-gradient(to bottom right, red , blue);
      background-image: linear-gradient(180deg, red, blue);
      background-image: linear-gradient(0deg, red, blue);
      background-image: linear-gradient(-90deg, red, blue);
      background-image: linear-gradient(to right, red,orange,yellow,green,blue,indigo,violet); <-- tipo arco-íris
      background-image: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1));

    • radial-gradient("shape(ellipse/circle) size(farthest-corner/closest-side/closest-corner/farthest-side)" at "position(center/porc_esquerda porc_direita)", "color 1", "color 2", ... , last color):

      • Ex:
      background-image: radial-gradient(red, green, blue);
      background-image: radial-gradient(red 5%, green 15%, blue 60%);
      background-image: radial-gradient(circle, red, yellow, green);
      background-image: radial-gradient(closest-side at 60% 55%, blue, green, yellow, black);
      background-image: radial-gradient(farthest-side at 60% 55%, blue, green, yellow, black);

    • none: Nenhuma imagem de background será mostrada. É o padrão.

  • background-origin

    Especifica onde a imagem(ns) de background está(ão) posicionadas.

    • { background-origin: padding-box | border-box | content-box | ;}

    • padding-box: Valor padrão. A imagem de background começa no lado superior esquerdo, na borda do padding.
    • border-box: A imagem de background começa no lado superior esquerdo, mas já por trás da borda, ultrapassando o padding.
    • content-box: A imagem de background começa no lado superior esquerdo do conteúdo. Ex: se fosse um texto, estaria dentro do bloquinho do texto, começando pelo canto superior esquerdo.

    Ex: Defina duas background-images em um elemento div, selecione uma para ser o background no cantinho esquerdo na borda do padding e outra para o cantinho esquerdo do conteúdo.
    {
    background: url(imagem_1.jpg), url(imagem_2.jpg);
    background-origin: padding-box, content-box;
    }
    Importante ficar atento na ordem das imagens e dos parâmetros passados no bg-origin.

  • background-position

    Define a posição inicial de uma imagem de background.

    • { background-position: (left,right,center) (top, center, bottom) faz o chuveirinho e escolhe o par que quiser | X% Y% | Xpos Ypos ;}

    • left top/left center/left bottom/right top/.../center bottom: Se for passado apenas um parâmetro, o outro será considerado "center".
    • X% Y%: O primeiro valor é na horizontal (eixo x) e o segundo é na vertical (eixo y). O canto superior esquerdo está em 0% 0% (também é o valor default). O canto inferior direito está em 100% 100%. Se passar apenas um parâmetro, o outro será considerado 50%.
    • Xpos Ypos: O primeiro valor é horizontal e o outro vertical. O canto superior esquerdo é 0px 0px (pode usar pixel ou qualquer outra unidade CSS, tipo rem). Se especificar apenas um valor, o outro será considerado 50%. Dá para misturar % com positions.

  • background-repeat

    Define como uma imagem de background será repetida.

    • { background-repeat: repeat | repeat-x | repeat-y | no-repeat | space | round | inicial | inherit ;}

    • repeat: Valor padrão. A imagem de fundo vai se repetir tanto horizontal quanto vertical. A última imagem será cortada se não couber.
    • repeat-x: A imagen de fundo vai se repetir apenas horizontalmente.
    • repeat-y: A imagem de fundo vai se repetir apenas verticalmente.
    • no-repeat: A imagem de fundo não vai se repetir. Só aparece uma.
    • space: A imagem de fundo será repetida o máximo que der sem ser cortada. A primeira e a última imagens são fixadas em ambos os lados do elemento, e os espaços em branco são distribuídos uniformememente entre as imagens.
    • round: A imagem de fundo será repetida e comprimida ou esticada para preencher o espaço (sem lacunas).

  • background-size

    Especifica o tamanho da(s) imagem(ns) de background.

    • { background-size: auto | length | cover | contain | initial | inherit ;}

    • auto: Valor padrão. A imagem de fundo é mostrada em seu tamanho original.

      • Ex: { background-size: auto ;}

    • length: Define a largura e altura da imagem de fundo. O primeiro valor é para a largura e outro para altura. Se for passado somente um valor, o outro será considerado "auto".

      • Ex: { background-size: 100px 150px ;}

    • percentage: Define a largura e altura da imagem por um par de porcentagens. O primeiro valor é para largura, o outro para altura. Se apenas um valor for passado, o outro será considerado "auto".

      • Ex: { background-size: 75% 100% ;}

    • cover: Redimensiona a imagem de fundo para cobrir todo o contêiner, mesmo que seja necessário esticar a imagem ou cortar um pouco uma das bordas.

      • Ex: { background-size: cover ;}

    • contain: Redimensiona a imagem de fundo para garantir que a imagem seja totalmente visível.

      • Ex: { background-size: contain ;}

  • background

    Define todas as propriedades de background em uma única declaração.

    • { background: bg-color bg-image position/bg-size bg-clip bg-attachment ;}

    Exemplo: body { lightblue url("img_tree.gif") no-repeat fixed center ;}

    Sintaxe em Javascript. Ex: object.style.background="red url(smiley.gif) top left no-repeat";

    Observação: se uma das propriedades na declaração abreviada for a propriedade bg-size, você deve usar uma / (barra) para separá-la da propriedade bg-position, por exemplo, plano de fundo: url (smiley.gif) 10px 20px / 50px 50px; resultará em uma imagem de fundo, posicionada a 10 pixels da esquerda, 20 pixels da parte superior e o tamanho da imagem será de 50 pixels de largura e 50 pixels de altura

    Nota: Se estiver usando várias fontes de background-image, mas também quiser um background-color, o parâmetro da cor de fundo precisa ser o último da lista.
