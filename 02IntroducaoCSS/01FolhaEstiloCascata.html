Como incluir um arquivo CSS na página HTML?


  • Ontem tem parênteses, é < >

  • Inline: (p style="...")...(/p) - desse modo não é recomendável, pois em caso de manutenção, seria necessário alterar parágrafo por parágrafo. existe um meio mais eficiente.

  • Interno: (style)...(/style) - pode passar os dados pela tag style, que fica no head. também não é muito recomendável, pois fica inviável quando trabalhando com muitas páginas

  • Externo: (link href="meusEstilos.css" rel="stylesheet") - É o meio mais usado, vinculando um arquivo CSS externo, podendo alterar dados em apenas uma página e ele modifica todas as outras

Seletores de elementos

novamente, no lugar de parênteses, são < >

  • HTML: (p) ... (/p)
  • CSS: p { ... ;}

Seletores de classes

Além das tags, podemos identificar componentes por meio de CLASSES. novamente, no lugar de parênteses, são < >

  • HTML: (p class="minha-classe") ... (/p)
  • CSS: .minha-classe { ... ;}

Seletores de ID

- Usar somente um id por componente, não repetí-los, identificadores são únicos.

- Além das tags e classes, podemos identificar componentes por meio de ID. novamente, no lugar de parênteses, são < >

  • HTML: (p id="meu-id") ... (/p)
  • CSS: #meu-id { ... ;}

fonts

  • font-family: Arial, Times, etc.
  • font-size: Tamanho
  • font-style: Pôr para itálico ou não: italic, normal, oblique
  • font-variant: normal, small-caps(minusculo), initial (configura para a propriedade padrão da fonte), inherit(herda propriedade de seus "parêntes")

text

  • text-align: Alinhar right, left ou center, justify
  • text-decoration: Se quer sublinhar ou não. "overline" "line-through" "underline" "none" (pra tirar sublinhado).
  • text-indent: Se temos um espaçamento entre o começo do parágrafo e o resto do testo
  • text-transform: Se desejo escrever tudo maiúsculo ou minúsculo. uppercase / lowercase / capitalize.
  • text-shadow: Para dar um efeito de sombra na fonte, ele recebe 4 parâmetros para formar a sombra: Horizontal Vertical Blur(tipo esfumaçado, também é em pixel) e a cor da sombra.

    • Exemplo: text-shadow: 3px 3px 5px red

  • font-weight: bold / normal / lighter / algum Nº inteiro "900" por exemplo.

RGB

  • #FFFFFF: Branco
  • #FF0000: Vermelho
  • #00FF00: Verde
  • #0000FF: Azul
  • #FFFF00: Amarelo
  • #FF00FF: Rosa
  • #00FFFF: Azul clarinho bonitinho

Atributos de cor: color(fonte) & background-color(fundo)
