Pseudo-classes

  • a:link{ ... } - É o link normal, sem ter sido visitado
  • a:visited{ ... } - É o link depois de visitado
  • a:hover{ ... } - É quando o mouse está em cima do link
  • a:active{ ... } - É o link no momento que foi clicado

  • a:hover DEVE vir depois do a:link and a:visited
  • a:active DEVE vir depois do a:hover

Exemplo:
a:link, a:visited {
background-color: #f44336;
color: white;
padding: 14px 25px;
}

a:hover, a:active {
background-color: red;
}

Isso é um link

List-Style

Além dos modelos de lista ordenada (ol) e não ordenadas (ul). temos os estilos

list-style-type: circle/none/square/upper-roman/decimal/decimal-leading-zero/lower-alpha

  • Tipo: circle
  • estilo: none
  • Tipo: square
  • Tipo: upper-roman
  • Tipo: decimal
  • Tipo: decimal-leading-zero
  • Tipo: lower-alpha

list-style-position: outside/inside/none

  • Position: outside
  • Position: inside
  • Position: none

list-style-image: url("link da imagem")/none/inicial

  • Imagem: url("link de imagem")
  • Imagem: none
  • Imagem: inicial

Passar os estilos em apenas uma linha

list-style: list-style-type list-style-position list-style-image ;
