Outros exemplos de como trabalhar com Margin:

Se usar "margin: auto ;" ele centraliza o elemento no meio da tela, mas só na horizontal. usando "margin: 30px auto ;" ele centraliza, e o primeiro valor é altura, faz o elemento descer um pouco.

p {
margin-top: 50px;
margin-right: 40px;
margin-bottom: 30px;
margin-left: 20px;
}

Mas também podemos escrever essas propriedades em uma única linha escrevendo apenas "margin", a ordem das dimensões segue o sentido horário, começando por cima.

p { margin: 50px 40px 30px 20px ;}

p { margin: (cima) (direita) (baixo) (esquerda) ;}

Segue essa ordem

Outros exemplos de como trabalhar com Padding:

p {
padding-top: 50px;
padding-right: 40px;
padding-bottom: 30px;
padding-left: 20px;
}

A ordem é a mesma que a do margin
p { padding: 50px 40px 30px 20px ;}
p { padding: (cima) (direita) (baixo) (esquerda) ;}

Abreviando margin/padding

Se passamos as dimensões em 3 ou menos direções. O próprio css assimila os outros valores com base nos que foram passados. Ex:

p { padding: 50px 40px 30px ;}

Não passamos um padding na direção esquerda, mas o próprio css assume o mesmo valor que foi passado em seu lado oposto (direita). Ou seja, o padding da esquerda fica com 40px.

Se passarmos menos de 3 valores, ele faz o mesmo, pega os mesmo valores das suas dimensões opostas que foram passados e os replica. Se passarmos apenas 50px para cima, ele vai adicionar esse valor para todo o resto (direita, baixo, esquerda).

Border

border-top/right/bottom/left: largura em pixels;

Podemos fazer também. Assim como no padding, margin, etc:

p { border-width: 50px(cima) 40px(direita) 30px(baixo) 20px(esquerda) ;}

p { border-color: red(cima) green(direita) blue(baixo) yellow(esquerda) ;}

p { border-style: dotted(cima) solid(direita) double(baixo) dashed(esquerda) ;}

p { border-radius: 15px(cima-esquerda) 50px(cima-direita) 30px(baixo-direita) 5px(baixo-esquerda) ;}

Alguns outros exemplos de como implementar bordas e efeitos:

h1 {
border-style: dotted;
boder-width: 5px;
border-color: red;
}

p {
border-top-style: dotted;
border-right-style: solid;
border-bottom-style: dotted;
border-left-style: solid;
}

p {
border-top-color: purple;
border-right-color: blue;
border-bottom-color: orange;
border-left-color: green;
}

p {
border-top-left-radius: 15px;
border-top-right-radius: 50px;
border-bottom-right-radius: 30px;
border-bottom-left-radius: 5px;
}

Border possui uma ordem em suas atribuições de tipos de valores:

p { border: dotted 5px red ;}

p { border: (style) (width) (cor) ;}

Exemplo: p { border-top: solid 10px blue ;}

Alguns valores para border-style:

  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value
  • inset - Defines a 3D inset border. The effect depends on the border-color value
  • outset - Defines a 3D outset border. The effect depends on the border-color value
  • none - Defines no border
  • hidden - Defines a hidden border
