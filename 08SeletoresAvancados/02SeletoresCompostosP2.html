Usando a pseudo classe "nth-child(n)". Esse seletor corresponde a cada elemento que é o N-ézimo filho, independentemente do tipo, de seu pai. É preciso específicar o elemento que deseja alterar, por tag ou classe "article:nth-child(número da posição que ele ocupa)" ou ".classe:nth-child(número da posição que ele ocupa)"

n pode ser um número, uma palavra-chave ou uma fórmula.

Exemplos

odd (ímpares) e even (pares) são palavras-chaves que podem ser usadas para corresponder a elementos filhos cujo índice é ímpar ou par (o índice do primeiro filho é 1). No exemplo abaixo, especificamos diferentes background colors para elementos parágrafos ímpares e pares.

      p:nth-child(odd) {
          background: red;
      }

      p:nth-child(even) {
          background: blue;
      }
