DOM(Document Object Model)

Esse assunto é mais para entender as hierarquias entre os componentes do documento HTML. Nesse exemplo, o Article é pai do H3 e dos parágrafos P, que por sua vez é pai do A, que é pai do B.
No momento de estilizar algo no css, é interessante saber quem está contido em cada grau de parentesco. Para esitilizar o A, faríamos algo do tipo "article p a", onde cada espaço serve para indicar que estamos indo mais a fundo nos graus de parentescos.

No lugar da gente querer selecionar todos os A que são filhos direto de P, isso quer dizer que mesmo que eu fizesse "article p > b", ele não faria nada, a propriedade que eu passasse não iria ser aplicada, pois não existe nenhum B filho direto de P. O filho de P é o A, que é pai de B. Podemos fazer da seguinte forma

Descendência

article p > a {color: red;}

No caso da gente querer indicar o irmão adjacente, ou seja, o irmão seguinte do elemento anterior, fazermos da seguinte forma e ele vai selecionar apenas o parágrafo que vem em seguida, dentro do article:

Irmão adjascente

article h3 + p {color: red;}

Considerando os exemplos da imagens, se considerássemos article p + p {color: red;} ele selecionaria os dois últimos parágrafos, pois ambos são Parágrafos que vem antes de outro Paráfrafo.

Podemos também selecionar todos os irmãos que vem DEPOIS de um elemento. Por exemplo, vamos selecionar todos os irmãos que vem depois do H3:

Próximos Irmãos

article h3 ~ p {color: red;}

SELETORES DE ATRIBUTOS

Essa é uma forma que podemos usar para selecionar um elemento específico quando temos vários desse tipo no nosso article/parágrafo/div/etc. Por exemplo, temos um article que dentro tem dois links, um com o atributo Title e ou sem:
<'a title=""> Com Title <'/a>
<'a>Sem Title<'/a>
Para selecionar apenas o link que possui Title, usamos colchetes e colocamos dentro dele o nome do atributo:

a[title] {color: red;}

No de dois links possuir o atributo Title, podemos referenciar aquele que desejamos, passando como referência o valor que foi colocado dentro do atributo:

<'img alt="Praia">
<'img alt="Ilha">

Para referenciar apenas uma, usamos:

img[alt="Ilha"] {padding: 20px;}

Podemos referenciar um ou vários elementos com base nos valores. Em caso de alguns valores se repetirem, pode-se usar isso como base para referenciar o elemento:

<'a href="..."> <'a>
<'a href="http://..."> <'a>
<'a href="http://www..."> <'a>

a[href^="http"] {color: red;}

Aqui ele vai atribuir essas propriedades para todos os links que no seu atributo HREF tem como valor uma URL começando com "http".

Também temos o inverso, referenciando Tags por meio de atributos que receberam valores parecidos.

Podemos referenciar elementos com base nos últimos caracteres dos valores que foram adicionados nos atributos. Por exemplo:

  • <'img src="...jpg">
  • <'img src="...png">
  • <'img src="...jpg">
Podemos referenciar as imagens que queremos com base no seu atributo SRC e nos seus últimos caracteres.

img[src$=".png"] {border: 3px solid red ;}

E selecionamos apenas as imagens que terminam com ".png".

Podemos usar tbm o :not()

Nesse caso, estamos querendo referenciar alguma coisa, exceto outra coisa. Exemplo:

Estamos referenciando todos elementos da página, menos os links (a)
:not(a) { color: red ;}

Estamos referenciando todos os parágrafos, menos aqueles que possuem a classe "destaque"
p:not(.destaque) { color: red ;}

Estamos referenciando todas as imagens, menos aquelas que possuem o atributo "alt"
img:not([alt]) { display: none ;}

Existem outros atributos que valem a pena serem pesquisados no w3schools.com e no developer.mozilla.org
