Imagens

Ao criar um site do zero, é importante separar uma pasta exclusivamente para as imagens usadas no site. Além de renomeá-las adequadamente, para facilitar sua linkagem. Em grandes sites, é comum separar as imagens em subpastas, pelo bem da organização. Imagens do tipo logos, botões, coisas desse tipo é bom colocar em uma pasta nomeada "interface". Imagens de produtos, poderia ficar em uma pasta chamada "products", imagens relacionadas à notícias, poderia ficar na pasta "news". Enfim, são alguns exemplos.

Tag "img" e os atributos src, alt, title:

Para adicionar uma imagem, usa-se a tag <'img> e dentro dela, usamos os atributos src="", e dentro das aspas colocamos a URL da imagem, que pode ser uma URL de alguma imagem na internet ou o caminho de alguma imagem dentro do seu pc. Temos o atributo alt="" que serve para colocar uma descrição na imagem, caso a mesma não carregue na página (é semântico) Vale lembrar que se a imagem não tem nenhum significado para a página, é só pra fim de estética, o atributo alt deve ser passado como uma string vazia, sem descrição. Temos também title="" que serve para fornecer alguma informação adicional à respeito da imagem. A maioria dos navegadores vão mostrar esse atributo como uma dica quando o mouse estiver em cima da imagem

Temos outros atributos que podemos usar dentro da tag <'img>, que são os atributos de dimensões width="número" e height="número" Esses atributos servem para especificar a largura e altura da imagem mostrada na tela, respectivamente. Lembrando que essas dimensões podem ser feitas tanto no arquivo CSS, quanto dentro da própria tag imagem.

Temos o atributo align que recebe como parâmetro os valores left e right. Esse atributo é normalmente usado para indicar como outras partes da página flui em volta de uma imagem. Por exemplo, se alinhado à esquerda, a imagem ficará à esquerda e o texto à direita. O texto continua fluindo normalmente em volta da imagem.

3 regras para criar imagens

1 - Salvar a imagem no formato certo. Prestar atenção no formato da imagem que irá salvar.

2 - Salvar imagens no seu tamanho correto. Salvar a imagem com o mesmo tamanho que será mostrada no site, para evitar que fique distorcida ou esticada.

3 - Usar a resolução correta de imagem. Ficar atento à isso. A maioria dos computadores apresentam sites em apenas 72 pixels por polegada. Salvar imagens maiores que o necessário irá atrazar o carregamento da imagem na página.

Formatos de Imagens

JPEG

Sempre que tiver muitas cores em uma imagem, tipo uma fotografia, deve-se usar JPEG.

GIF ou PNG

Quando salvar uma que possui poucas cores ou grandes áreas com a mesma cor, tipo logos, ilustrações que possuem a mesma cor preenchendo quase tudo, com cores chapadas, vale a pena salvar a imagem como gif (ou PNG).

Dimensões de Imagens

A imagem que se deseja usar no website, deve ser salva com as mesmas dimensões que será apresentada no site. Dá para dimensioná-las com o paint, photoshop ou outro editor de imagens. Reduzindo o tamanho das imagens melhora no carregamento/download da imagens no site.

Não dá para aumentar o tamanho de uma imagem significativamente sem afetar a qualidade da imagem. Se sua imagem tem uma resolução de 100x50 e tu aumenta em 300%, vai ficar uma bosta. A imagem vai ficar borrada ou pixelada, com aqueles bloquinhos horríveis.

Apenas alguns tipos de imagens podem ser cortadas sem perder informações valiosas. Detalhe: imagens carregam muito mais informações que a posição de cada pixel que no fim formam uma imagem. Pesquisar mais sobre!

Cortando Imagens

Quando pesquisar ou cortar imagens, procure por imagens em seu formato e resolução adequado, ficar atento se a imagem está em formato retrato ou paisagem.

Resolução de Imagem

Imagens criadas para web devem ser salvas com uma resolução de 72ppi (pixels per inch). Quanto maior a resolução da imagem, maior o tamanho do arquivo.

JPGs, GIFs e PNGs percetem ao tipo de imagem conhecido por bitmap(pesquisar mais sobre!). Elas são feitas de vários quadrados em miniaturas (pixels). A resolução de uma imagem é medida pelo número de pixels contidos dentro de uma polegada quadrada.

Imagens impressas, tipo revistas ou livros, são feitos de pequenos circulos "dots". Essas imagens normalmente são impressas com a resolução de 300 dots per inch (dpi). Devido ao fato dos monitores de computador serem limitados a 72ppi, usando imagens na web com resolução maior que isso não vai melhorar a qualidade de imagem.

Imagens Vetoriais

Imagens vetoriais se diferenciam das bitmaps por meio da sua resolução independente. Elas são normalmente criadas em programas tipo o Adobe Illustrator. Quando a imagem tem umas linhas desenhadas, tipo um logo, ilustração ou diagrama, são melhores quando estão em formato vetorial.

As imagens vetoriais são criadas colocando pontos em uma grade e desenhando linhas entre esses pontos. Uma cor pode então ser adicionada para "preencher" as linhas que foram criadas.

As vantagens de criar desenhos de linhas em formato vetorial e que dá para aumentar ou diminuir as dimensões das imagens sem perder a qualidade. O método recente de usar imagens vetoriais para monitores em websites envolve em salvar uma versão bitmap da imagem vetorial original e usando ela.

Scalable Vector Graphics(SVG) são um novo formado usando para mostrar imagens vetoriais diretamente da web, eliminando a necessiadade de criar uma versão da imagem em bitmap.

Animated GIFs

GIFs animados mostram vários quadros de uma imagem em sequência, embora possa ser usada para criar simples animações. GIF é basicamente um slide que passa bem rápido várias imagens e que dão a impressão de movimento (é basicamente um vídeo, mas sem o áudio). Alguns programas, tipo Photoshop permite a criação de GIFs, além de alguns websites.

Vale lembrar que cada frame adicionado vai aumentar o tamanho da sua imagem GIF e o tempo de carregamento da imagem. Nem todo mundo gosta de ficar esperando o carregamento, portanto, use GIFs com moderação. Esse formato é mais recomendado para coisinhas simples, tipo aqueles carregamentos de vídeos, aquela bolinha que ficava girando no antigo youtube, por exemplo.

Tranparência

Criar uma imagem que é parcialmente transparente (aquelas imagens sem fundo) para a web envolve em selecionar dois formatos: Transparent GIF ou PNG

Se a a parte transparente da imagem tem bordas retas que são 100% transparentes, dá para salvar a imagem como um GIF com a opção de transparência selecionada.

Se a parte transparente da imagem tem bordas diagonais ou redondas or se deseja-se uma transparência meio opaca ou uma sombra no fundo, então é importante salvar como PNG.

Imagens PNGs transparentes não são suportadas em navegadores mais antigos, mas dá para burlar isso com javascript.

Examinando Imagens na Web

Ao atualizar um site, é importante verificar se o tamanho da imagem do site antes de criar uma nova e substituí-la. Isso pode ser feito clicando com o botão direito do mouse na imagem e abrí-la em uma nova janela.

HTML5: Figure e Figure Caption

Imagens normalmente vêm com legendas e isso foi introduzido no HTML 5 com a tag <'figure> que serve para colocar uma legenda com uma descrição da imagem. Dá para adicionar mais de uma imagem dentro da mesma tag <'figure> desde que elas compartilhem da mesma legenda.

A tag <'figcaption> foi adicionada ao HTML 5 em desejo de permitir que o autores do website a adicionarem uma legenda em uma imagem.

A tag "figcaption" fica dentro da tag "figure", lembrando!!!
