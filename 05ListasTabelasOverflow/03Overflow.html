Overflow

Nota: A propriedade overflow só funciona para elementos de bloco com uma altura especificada.

Essa propriedade Overflow, serve para tratar o conteúdo que excede determinado elemento. Ela também serve para controlar o que acontece ao conteúdo que é grande demais para encaixar em uma área.

Visible
Valor padrão. O excesso de conteúdo(overflow) não é cortado. O conteúdo é renderizado fora da caixa do elemento.

Exemplo: div { width: 200px; height: 50px; background-color: #eee; overflow: visible ;}

Hidden
Aqui o excesso de conteúdo(overflow) é cortado, e o resto do conteúdo cortado é escondido.

Exemplo: div { overflow: hidden ;}

Scroll
Selecionando Scroll, o excesso de conteúdo é cortado e será adicionado uma scrollbar dentro da caixa. Perceba que isso vai adicionar uma scrollbar tanto na vertical quanto na horizontal, mesmo que não precise.

Exemplo: div { overflow: scroll ;}

Auto
É semelhante ao Scroll, mas adiciona uma scrollbar se necessário. Engloba todo o conteúdo em excesso.

Exemplo: div { overflow: auto ;}
